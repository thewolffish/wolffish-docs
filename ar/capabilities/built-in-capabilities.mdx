# القدرات المضمَّنة

تأتي هذه القدرات ضمن مساحة العمل الافتراضية عند إنشاء Wolffish لمجلد `~/.wolffish/workspace/` عند التشغيل الأول.

## shell

**النوع:** إضافة

تغلّف `child_process` في Node.js لتنفيذ أوامر الشل. توفر أداة `shell_exec` مع معامل `command` (سلسلة نصية) ومعامل اختياري `background` (قيمة منطقية) للعمليات المنفصلة.

تحجب أنماط الخطر الأوامر المدمرة مثل `rm -rf /` و`sudo rm` و`chmod 777`. تتطلب أنماط التأكيد موافقة لعمليات مثل `npm install` و`git push` و`docker rm`.

## filesystem

**النوع:** إضافة

تغلّف `fs` في Node.js لعمليات الملفات. توفر ثلاث أدوات:

- `file_read` — قراءة محتوى الملف
- `file_write` — كتابة محتوى إلى ملف (ينشئ أو يستبدل)
- `file_patch` — تطبيق تعديلات موجَّهة على ملف موجود

## git

**النوع:** مهارة نقية (بدون إضافة)

توفر تعليمات git لنموذج اللغة عبر SKILL.md. يستخدم نموذج اللغة `shell_exec` لتشغيل أوامر git. تشمل التعليمات استخدام تنسيق الالتزام التقليدي، وتشغيل `git status` أولًا، وعرض التغييرات قبل الالتزام.

## introspect

**النوع:** إضافة

قدرة الوعي الذاتي. توفر ثلاث أدوات:

- `wolffish_status` — أعداد المهام الحالية، ووقت التشغيل، والمزودين النشطين
- `wolffish_performance` — معدلات النجاح من ملاحظات basalganglia
- `wolffish_memory` — ملخصات الحلقات الأخيرة وإحصائيات الذاكرة

عندما تسأل "كيف حالك؟" أو "ماذا كنت تفعل؟"، يستدعي نموذج اللغة هذه الأدوات لقراءة حالته الخاصة.

## تعديل القدرات المضمَّنة

كل القدرات المضمَّنة هي مجرد مجلدات في مساحة عملك. يمكنك تحرير ملفات SKILL.md الخاصة بها لتغيير السلوك، أو إضافة أنماط خطر جديدة، أو تعديل تعليمات الأدوات. تسري التغييرات على الرسالة التالية — لا حاجة لإعادة التشغيل.

<Warning>
  إذا حذفت قدرة مضمَّنة، فلن يُعيد Wolffish إنشاءها تلقائيًا. احتفظ بنسخة احتياطية أو استخدم git لإدارة إصدار مساحة عملك.
</Warning>
