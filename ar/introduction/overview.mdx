# نظرة عامة

وولف فيش هو تطبيق ذكاء اصطناعي شخصي لسطح المكتب يعمل محليًا أولًا ومدعوم بملفات Markdown، تم بناؤه باستخدام Electron. يعمل على macOS وWindows وLinux، ويُسقط بنية الدماغ البشري على خطّ معالجة برمجي حتمي، حيث تتولى 15 وحدة تشغيل وقت التنفيذ كل منها وظيفة واحدة محددة — توجيه المدخلات، وبناء السياق، وإدارة الذاكرة، وتنفيذ الأدوات، وفرض الأمان، والتعلم من النتائج.

## التثبيت

macOS / Linux / Windows:

```bash
curl -fsSL https://releases.wolffi.sh/install.sh | sh
```

Windows (PowerShell):

```powershell
irm https://releases.wolffi.sh/install.ps1 | iex
```

أو حمّل أحدث إصدار مباشرة من [wolffi.sh](https://wolffi.sh/).

## الفكرة الجوهرية

**Markdown هو المصدر الموثوق لكل سلوكيات الوكيل.** كود TypeScript ليس إلا أنابيب توصيل خالصة — يقرأ Markdown، ويستدعي نماذج اللغة الكبيرة، وينفذ الأدوات، ثم يكتب Markdown مرة أخرى. لتغيير ما يفعله الوكيل، تُحرّر ملف Markdown، لا الكود. يقرأ نموذج اللغة هذه التعليمات في وقت التشغيل، فيتطور السلوك دون الحاجة إلى إعادة بناء.

## كيف يعمل

يتواصل وولف فيش مع نماذج اللغة الكبيرة عبر تسعة مزودين سحابيين — DeepSeek وXiaomi MiMo وQwen (علي بابا كلاود) وKimi (Moonshot AI) وMiniMax وxAI (Grok) وStepfun وAnthropic (Claude) وOpenAI (GPT) — بالإضافة إلى Ollama للنماذج المحلية. جميعهم اختياريون. تختار نموذج **الدماغ** لديك صراحةً في الإعدادات — الذي تختاره هو من يعمل. لا يوجد تدرّج أو احتياطي تلقائي. اختياريًا، فعّل **وضع التنسيق** لإضافة نموذج **عامل** ثانٍ للعمل المتوازي. نوصي بالبدء بـ DeepSeek أو MiMo — يتعاملان مع المهام الوكيلية المعقدة بشكل جيد بجزء من التكلفة. بدّل إلى Anthropic أو OpenAI عند الحاجة فقط.

كل شيء يعيش في مجلد واحد: `~/.wolffish/`. احذفه وستعود إلى الإعدادات الافتراضية للمصنع. لا توجد عمليات خلفية على مستوى النظام، ولا مهام cron، ولا إدخالات في السجل، ولا ملفات تكوين متناثرة.

```
رسالة المستخدم → بناء السياق → استدعاء نموذج اللغة → تنفيذ الأدوات → الذاكرة → الرد
```

يتم التعامل مع كل خطوة بواسطة وحدة دماغية مخصصة، وكل خطوة قابلة للفحص عبر ملفات Markdown.

## لمن هو موجَّه؟

وولف فيش موجّه لأي شخص يريد وكيل ذكاء اصطناعي شخصي يفهمه بالكامل، ويستخدمه بفعالية، ويوسّعه بموثوقية. لا تحتاج أن تكون مطوراً — تحتاج فقط أن تفهم كيف يعمل. كل شيء عبارة عن ملفات Markdown عادية يمكنك قراءتها وتعديلها وتصحيح أخطائها. إن كنت تريد أن تملك أدوات الذكاء الاصطناعي بدلاً من استئجارها، فإن وولف فيش لك.
