# MiniMax

```
POST https://api.minimaxi.chat/v1/chat/completions
```

يستخدم تدفق SSE بتنسيق أدوات متوافق مع OpenAI. يدعم محتوى الاستدلال.

الأفضل لـ: أحمال العمل المكثفة بالاستدلال، توليد الأكواد، سير العمل الوكيلي.

<Note>
  MiniMax مزوّد صيني جيد، لكن DeepSeek وMiMo يبقيان أرخص وأكثر قدرة لمعظم أحمال العمل. فكّر بـ MiniMax إذا أردت تنويع المزودين أو كانت لديك احتياجات محددة تخدمها تشكيلة نماذجه جيدًا.
</Note>

## الحصول على مفتاح API

1. اذهب إلى [platform.minimaxi.chat](https://platform.minimaxi.chat)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← MiniMax
