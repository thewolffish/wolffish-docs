# Qwen (علي بابا كلاود)

```
POST https://dashscope-intl.aliyuncs.com/compatible-mode/v1/chat/completions
```

يستخدم تدفق SSE بتنسيق أدوات متوافق مع OpenAI. يدعم الرؤية (صور base64) ومحتوى الاستدلال.

**يقدم Qwen واحدة من أوسع تشكيلات النماذج** — من Qwen 3.5 Flash فائق الرخص بسعر $0.06/$0.24 لكل مليون رمز إلى Qwen 3.7 Max الحدودي. جميع نماذج Qwen3+ تدعم ثلاثة أوضاع استدلال (بدون، عالي، أقصى) وسياق يصل إلى 1M. نموذج Qwen3 Coder Plus مخصص لمهام توليد الشفرة البرمجية.

الأفضل لـ: سير العمل الوكيلي الاقتصادي، توليد الشفرة البرمجية، المهام متعددة اللغات.

<Note>
  Qwen 3.5 Flash هو أحد أرخص النماذج القادرة على الاستدلال — بسعر $0.06/$0.24 لكل مليون رمز مع سياق 1M، وهو أرخص بكثير من DeepSeek V4 Flash مع دعم كامل لأوضاع الاستدلال. ممتاز للمهام عالية الحجم.
</Note>

## الحصول على مفتاح API

1. اذهب إلى [qwencloud.com](https://www.qwencloud.com)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← Qwen
