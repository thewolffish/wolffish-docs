# Xiaomi MiMo

```
POST https://api.xiaomimimo.com/v1/chat/completions
```

يستخدم تدفق SSE بتنسيق أدوات متوافق مع OpenAI. يدعم الرؤية (صور base64) ومستندات PDF.

**MiMo هو مزوّد سحابي اقتصادي** يقدم نماذج تنافسية بجزء من تكلفة Anthropic وOpenAI. بعد [تخفيض Xiaomi الدائم للأسعار بنسبة تصل إلى 99%](https://www.studioglobal.ai/ms/discover/answers/what-prompted-xiaomi-to-cut-its-mimo-v2-5-6a179fe3f9dbfce068be8b7e) — لتطابق أسعار DeepSeek V4 Pro — يقدم MiMo V2.5 Pro استدلالًا قويًا ودعمًا متعدد اللغات بتكلفة أقل بكثير من النماذج الحدودية.

الأفضل لـ: سير العمل الوكيلي الاقتصادي، المهام متعددة اللغات، سلاسل أدوات متعددة الخطوات، والأتمتة عالية الحجم.

<Note>
  يتعامل MiMo مع المهام الوكيلية المعقدة بشكل جيد — بما في ذلك سلاسل أدوات طويلة متعددة الخطوات، وسير عمل البحث، وتوليد الأكواد، والأتمتة المستقلة. نوصي بالبدء بـ MiMo والترقية إلى **Anthropic** أو **OpenAI** فقط إذا لم يكن التنفيذ موثوقًا بما يكفي لسير عمل محدد، أو إذا احتجت **computer-use** (التفاعل مع الشاشة)، الذي يدعمه Anthropic فقط. انظر [اختيار مزوّد](/ar/configuration/providers#choosing-a-provider) للتفاصيل.
</Note>

## الحصول على مفتاح API

1. اذهب إلى [platform.xiaomimimo.com](https://platform.xiaomimimo.com)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← Xiaomi Mimo

MiMo وDeepSeek **أرخص بـ 10–25 ضعفًا** من Anthropic وOpenAI ويتعاملان مع المهام الوكيلية المعقدة بشكل جيد. ابدأ هنا — ارتقِ فقط إذا لم ينفذ سير عمل محدد بموثوقية، أو إذا احتجت computer-use (Anthropic فقط).

## متى تستخدم MiMo

**مناسب — معظم سير العمل:**
- سلاسل أدوات متعددة الخطوات (بحث ملفات ← قراءة ← تحرير ← إيداع)
- سير عمل بحث ← تلخيص ← كتابة
- الأتمتة المجدولة ومعالجة المهام اليومية
- المهام متعددة اللغات — يتمتع MiMo بدعم قوي للصينية والإنجليزية واللغات المتعددة
- صياغة البريد وتوليد التقارير واستخراج البيانات
- الوكلاء المحادثيون والأسئلة والأجوبة
- أحمال العمل عالية الحجم حيث تهم تكلفة الرمز

**فكّر بالطبقة المتميزة إذا:**
- احتجت computer-use / التفاعل مع الشاشة (يتطلب Anthropic)
- جربت سير عمل على MiMo والتنفيذ لم يكن موثوقًا بما يكفي — ارتقِ إلى Anthropic أو OpenAI لتلك المهمة المحددة

<Tip>
  ابدأ بـ MiMo أو DeepSeek كمزوّدك الأساسي وجرّب سير عملك الفعلي. معظم المستخدمين يجدون أنهم نادرًا ما يحتاجون للطبقة المتميزة. تدرج Wolffish يمكنه الانتقال تلقائيًا إلى Anthropic أو OpenAI عند ضبطهما.
</Tip>
