# xAI (Grok)

```
POST https://api.x.ai/v1/chat/completions
```

يستخدم تدفق SSE بتنسيق أدوات متوافق مع OpenAI. يدعم الرؤية (صور base64) ومحتوى الاستدلال.

**يقدم xAI نماذج Grok ذات قدرات استدلال قوية وأسعار تنافسية.** Grok 4.3 هو النموذج الرائد بسياق 1M ودعم الرؤية وثلاثة أوضاع استدلال — يقع بين المزودين الصينيين الاقتصاديين والطبقة الغربية المتميزة من حيث السعر. نموذج Grok Build الأحدث مصمم خصيصًا لتوليد الشفرة البرمجية بسعر جذاب.

الأفضل لـ: سير العمل المكثف بالاستدلال، توليد الشفرة البرمجية، مهام الرؤية، والأتمتة الوكيلية.

## الحصول على مفتاح API

1. اذهب إلى [console.x.ai](https://console.x.ai)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← xAI

<Note>

</Note>

<Tip>
  Grok 4.3 بسعر تنافسي مقارنة بـ Anthropic و OpenAI مع تقديم سياق 1M ودعم الرؤية. إذا كنت تحتاج مزوّدًا بطبقة متوسطة باستدلال قوي وأرخص من Claude أو GPT، فإن xAI خيار ممتاز.
</Tip>
