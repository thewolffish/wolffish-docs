# Anthropic (Claude)

```
POST https://api.anthropic.com/v1/messages
```

يستخدم تدفق SSE. تصل استدعاءات الأدوات ككتل محتوى `tool_use`.

الأفضل لـ: الاستدلال المعقد، واتباع التعليمات التفصيلية، والاستخدام الدقيق للأدوات، وcomputer-use (التفاعل مع الشاشة).

<Note>
  Anthropic هو المزوّد الوحيد الذي يدعم **computer-use** (التفاعل مع الشاشة). إن احتجت أن يقود Wolffish متصفحًا أو واجهة سطح مكتب، تحتاج مفتاح Anthropic.
</Note>

## الحصول على مفتاح API

1. اذهب إلى [console.anthropic.com](https://console.anthropic.com)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← Anthropic
