# Stepfun

```
POST https://api.stepfun.ai/v1/chat/completions
```

يستخدم تدفق SSE بتنسيق أدوات متوافق مع OpenAI. يدعم الرؤية (صور base64) ومحتوى الاستدلال.

**نماذج Step-3 تستدل في كل طلب** — لا يوجد مفتاح لتعطيل التفكير. تُحسب رموز الاستدلال ضمن ميزانية الإكمال، فيوازن النموذج تلقائيًا بين عمق التفكير وطول الإخراج. هذا يجعل Stepfun خيارًا مباشرًا عندما تريد استدلالًا دائمًا بدون إدارة إعدادات الأوضاع.

الأفضل لـ: المهام المكثفة بالاستدلال، أحمال العمل التي تتطلب تفكير النموذج دائمًا.

## الحصول على مفتاح API

1. اذهب إلى [platform.stepfun.ai](https://platform.stepfun.ai)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← Stepfun
