# OpenAI (GPT)

```
POST https://api.openai.com/v1/chat/completions
```

يستخدم تدفق SSE. تصل استدعاءات الأدوات ككائنات `function_call`.

الأفضل لـ: المهام العامة، والمعرفة الواسعة، والردود السريعة.

## الحصول على مفتاح API

1. اذهب إلى [platform.openai.com](https://platform.openai.com)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في Wolffish ← الإعدادات ← النماذج ← OpenAI
