# Kimi (Moonshot AI)

```
POST https://api.moonshot.ai/v1/chat/completions
```

يستخدم تدفق SSE بتنسيق أدوات متوافق مع OpenAI. يدعم الرؤية (صور base64) ومحتوى الاستدلال.

الأفضل لـ: سير العمل الوكيلي، المهام ذات السياق الطويل، أحمال العمل المكثفة بالاستدلال، والأتمتة اليومية الفعّالة من حيث التكلفة.

## الحصول على مفتاح API

1. اذهب إلى [platform.moonshot.ai](https://platform.moonshot.ai)
2. سجّل حسابًا أو سجّل الدخول
3. انتقل إلى **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في وولف فيش ← الإعدادات ← النماذج ← Kimi
