# إخبار وولف فيش عنك

يقع في `brain/identity/user.md`، ويخبر هذا الملف وولف فيش عنك. كلما قدمت سياقًا أكثر، أصبحت ردود الوكيل أكثر صلة وتخصيصًا.

## ماذا تُضمِّن

- اسمك ودورك
- الحزمة التقنية والأدوات التي تستخدمها
- المشاريع التي تعمل عليها
- التفضيلات (نمط الكود، اتفاقيات الالتزام، إلخ)
- أي شيء يساعد الوكيل على فهم سياقك

## مثال

```markdown
# User Profile

## About Me
- Name: Younes
- Role: Senior Full-stack Engineer
- OS: macOS
- Editor: VS Code

## Tech Stack
- TypeScript, React, Node.js, Electron
- Tailwind CSS, Vite
- Git with conventional commits

## Current Projects
- Wolffish: Local-first AI desktop agent
- Using npm, not yarn or pnpm

## Preferences
- Prefer concise code with clear variable names
- Always use TypeScript strict mode
- Prefer explicit imports over barrel exports
- Test with Vitest
```

## كيف يُستخدم

يُضمَّن `user.md` في قسم الهوية من نافذة السياق (تخصيص ميزانية 15%). لا يصفّيه RAS — يُضمَّن دائمًا لأن سياق الهوية ذو صلة دائمًا.

يستخدم نموذج اللغة هذه المعلومات لتخصيص الردود: اقتراح مدير الحزم المناسب، واستخدام أنماط الكود المفضلة لديك، وفهم سياق أسئلتك دون الحاجة إلى شرح إعدادك في كل مرة.
