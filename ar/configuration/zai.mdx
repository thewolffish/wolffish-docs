# Z.ai (Zhipu GLM)

```
POST https://api.z.ai/api/paas/v4/chat/completions
```

تدفق SSE متوافق مع OpenAI مع استدعاء الأدوات. يُبثّ الاستدلال بصيغة `reasoning_content`، ويُبدَّل التفكير عبر `thinking: { type }`، وتُبلَّغ إصابات ذاكرة التخزين المؤقت ضمن `usage.prompt_tokens_details.cached_tokens` — التنسيق نفسه المستخدم في Kimi.

**تفكير GLM ثنائي** — مُفعّل أو مُعطّل، دون مستويات جهد. أي وضع تفكير غير *none* يُفعّل الاستدلال، و*none* يُعطّله. كما يوفّر GLM-5.2 **نافذة سياق بمليون رمز**، وهي الأكبر في تشكيلة Z.ai.

الأفضل لـ: العمل الوكيلي الموفّر للتكلفة، وسير العمل طويل السياق (GLM-5.2)، والمهام التي تتطلب الرؤية. نماذج GLM متعددة الاستخدامات وقوية في سلاسل الأدوات والبرمجة بأسعار الفئة الاقتصادية.

## الحصول على مفتاح API

1. اذهب إلى [z.ai](https://z.ai)
2. سجّل حسابًا أو سجّل الدخول
3. افتح **API Keys** وأنشئ مفتاحًا جديدًا
4. الصقه في وولف فيش ← الإعدادات ← النماذج ← Z.ai

## النماذج

| النموذج | السياق | الإخراج الأقصى | الأوضاع | إدخال / إخراج (لكل مليون رمز) | ملاحظات |
|---------|--------|----------------|---------|-------------------------------|---------|
| **glm-4.6** | 200K | 64K | تفكير مُفعّل/مُعطّل | $0.60 / $2.20 | موصى به. حصان عمل وكيلي موفّر. |
| glm-4.5-air | 128K | 64K | تفكير مُفعّل/مُعطّل | $0.20 / $1.10 | الأرخص. خفيف وسريع. |
| glm-5.2 | 1M | 64K | تفكير مُفعّل/مُعطّل | $1.40 / $4.40 | الرائد. أكبر سياق (مليون). |

يعرض وولف فيش كل نماذج دردشة GLM التي يصل إليها مفتاحك (`glm-4.5`، `glm-4.5-air`، `glm-4.6`، `glm-4.7`، `glm-5`، `glm-5-turbo`، `glm-5.1`، `glm-5.2`) — اختر أيًّا منها من منتقي النماذج.

<Note>
  التفكير في GLM **كل شيء أو لا شيء** — يُطابق وولف فيش كل وضع تفكير غير *none* إلى مُفعّل، و*none* إلى مُعطّل. لا توجد مستويات جهد منخفض/متوسط/عالٍ. استدعاء الأدوات مدعوم دون حد أقصى لعددها.
</Note>

<Note>
  الإخراج محدود لكل نموذج (glm-4.5/4.5-air عند ~98K، وglm-4.6+ عند ~131K)؛ ويطلب وولف فيش ميزانية إكمال موحّدة قدرها 64K لترك مساحة للمحادثة في السياق.
</Note>
