# التثبيت

Wolffish تطبيق Electron مستقل بالكامل. لا حاجة لـ Node.js أو Python أو Docker. فقط ثبّت وشغّل.

## المتطلبات المسبقة

<AccordionGroup>
  <Accordion title="Ollama (مطلوب)" defaultOpen>
    يتطلب Wolffish وجود Ollama للاستدلال المحلي على نماذج اللغة الكبيرة. بدونه، لن يكون للوكيل أي قدرة على التفكير أو الاستجابة. ثبّته من [ollama.com](https://ollama.com)، ثم عُد إلى Wolffish.

    هذا كل شيء — يتكامل Wolffish مباشرة مع Ollama وسيساعدك في اختيار وتنزيل نموذج عند التشغيل الأول. لا حاجة لأوامر في الطرفية.
  </Accordion>

  <Accordion title="مزودو السحابة (موصى بهم للتجربة الكاملة)">
    للمهام الوكيلية المعقدة والردود الأعلى جودة، أعدّ مفاتيح API لأحدهما أو كليهما:
    - **Anthropic (Claude)**: احصل على مفتاح API من [console.anthropic.com](https://console.anthropic.com)
    - **OpenAI (GPT)**: احصل على مفتاح API من [platform.openai.com](https://platform.openai.com)

    ستضبط هذه المفاتيح في إعدادات Wolffish بعد التثبيت. يحاول تسلسل LLM استخدام مزودي السحابة أولاً ثم يعود إلى Ollama. للمهام البسيطة يكفي Ollama وحده، لكن مزودي السحابة مطلوبون للتجربة الكاملة والمهام المعقدة متعددة الخطوات.

    <Note>
      يمكن لـ Wolffish العمل محلياً بالكامل بدون مزودي سحابة — لكن المهام الوكيلية تتطلب نماذج كبيرة عالية القدرة (70B+ معامل) مثل Llama 3 70B أو Qwen 2.5 72B أو DeepSeek-V2. تشغيل هذه يتطلب 48GB+ ذاكرة أو بطاقة رسومات متقدمة، وهو ما لا يملكه معظم الناس. مدعوم لكن غير موصى به لمعظم الأجهزة.
    </Note>
  </Accordion>

  <Accordion title="أذونات macOS">
    لقدرة computer-use، سيطلب macOS منك منح أذ��نات **الوصول** و**تسجيل الشاشة**. هذه مطلوبة فقط إذا أردت أن يتفاعل Wolffish مع شاشتك.
  </Accordion>
</AccordionGroup>

## التنزيل

<Tabs>
  <Tab title="macOS">
    نزّل أحدث ملف `.dmg` من [صفحة الإصدارات](https://github.com/thewolffish/wolffish-app/releases).

    1. افتح ملف `.dmg`
    2. اسحب Wolffish إلى مجلد التطبيقات
    3. شغّل Wolffish من مجلد التطبيقات

    <Note>
      عند التشغيل الأول، قد يحذّر macOS من مطور غير معرَّف. انقر بزر الفأرة الأيمن على التطبيق واختر "فتح" لتجاوز ذلك.
    </Note>
  </Tab>

  <Tab title="Windows">
    نزّل أحدث مثبّت `.exe` من [صفحة الإصدارات](https://github.com/thewolffish/wolffish-app/releases).

    1. شغّل مثبّت NSIS
    2. اتبع معالج الإعداد
    3. شغّل Wolffish من قائمة ابدأ

    جميع الأذونات تُمنح تلقائياً على Windows.
  </Tab>

  <Tab title="Linux">
    نزّل أحدث `.AppImage` من [صفحة الإصدارات](https://github.com/thewolffish/wolffish-app/releases).

    ```bash
    chmod +x Wolffish-*.AppImage
    ./Wolffish-*.AppImage
    ```

    <Warning>
      X11 مطلوب. Wayland غير مدعوم حالياً لقدرة computer-use.
    </Warning>
  </Tab>
</Tabs>

## ما المضمّن

لا تحتاج لتثبيت أي شيء آخر. Electron يضمّن Node.js وChromium داخلياً. يأتي التطبيق مع:

- بيئة التشغيل (Node.js + Chromium)
- أكثر من 20 قدرة مدمجة
- SQLite (فهرس البحث FTS5)
- جميع محولات القنوات (سطح المكتب، Telegram، WhatsApp)

## المتطلبات حسب نظام التشغيل

| | macOS | Windows | Linux |
|---|---|---|---|
| **طريقة التثبيت** | `.dmg` ← سحب للتطبيقات | مثبّت NSIS (`.exe`) | AppImage (تفعيل الصلاحية، تشغيل) |
| **هل Node.js مطلوب؟** | لا | لا | لا |
| **تبعيات التشغيل** | لا شيء | لا شيء | لا شيء |
| **للذكاء المحلي** | Ollama (مطلوب) | Ollama (مطلوب) | Ollama (مطلوب) |
| **للذكاء السحابي** | مفتاح API (Anthropic أو OpenAI) | مفتاح API | مفتاح API |
| **الأذونات** | الوصول + تسجيل الشاشة | تلقائية | X11 مطلوب |

أدوات النظام مثل `ffmpeg` و`git` **ليست متطلبات مسبقة**. عندما تحتاج قدرة لأداة، يكتشف Wolffish أنها مفقودة ويعرض تثبيتها لك (عبر مدير حزم نظامك)، بموافقتك. الوكيل يدير التبعيات أثناء التشغيل.

## البناء من المصدر

<Warning>
  البناء من المصدر مخصص للمساهمين فقط. المستخدمون العاديون يجب أن يستخدموا المثبّتات الجاهزة أعلاه.
</Warning>

```bash
git clone https://github.com/thewolffish/wolffish-app.git
cd wolffish-app
npm install
npm run dev
```

يتطلب البناء من المصدر Node.js 20+ وnpm. الوحدات الأصلية (better-sqlite3) تُجمّع أثناء `npm install` عبر `electron-rebuild`.

## التحقق من التثبيت

عند التشغيل الأول، ينشئ Wolffish المجلد `~/.wolffish/workspace/` مع ملفات التكوين الافتراضية. ستوجّهك عملية الإعداد خلال اختيار نموذج والبدء.

للتحقق من أن كل شيء يعمل:
1. اتبع توجيهات الإعداد لاختيار وتنزيل نموذج
2. أرسل رسالة تجريبية مثل "مرحباً، ماذا يمكنك أن تفعل؟"
3. ينبغي أن ترى رداً متدفقاً

إذا لم يُكتشف Ollama، سيرشدك Wolffish خلال تثبيته.

## إلغاء التثبيت

جميع بيانات المستخدم موجودة في `~/.wolffish/`. لإلغاء التثبيت بالكامل:
1. احذف التطبيق
2. أزل مجلد البيانات: `rm -rf ~/.wolffish/`
