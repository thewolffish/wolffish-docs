# بلا اعتماديات، فقط ثبّت

وولف فيش تطبيق Electron مستقل بالكامل. لا حاجة لـ Node.js أو Python أو Docker. فقط ثبّت وشغّل.

## المتطلبات المسبقة

<AccordionGroup>
  <Accordion title="مزوّد LLM (مطلوب واحد على الأقل)" defaultOpen>
    يحتاج وولف فيش مزوّد LLM واحد على الأقل للتفكير والاستجابة. يمكنك استخدام **مزودي السحابة**، أو **نماذج محلية عبر Ollama**، أو كليهما:

    - **DeepSeek (V4 Pro)** ⭐ موصى به — احصل على مفتاح API من [platform.deepseek.com](https://platform.deepseek.com)
    - **Anthropic (Claude)**: احصل على مفتاح API من [console.anthropic.com](https://console.anthropic.com)
    - **OpenAI (GPT)**: احصل على مفتاح API من [platform.openai.com](https://platform.openai.com)
    - **Ollama (محلي)**: ثبّته من [ollama.com](https://ollama.com) للاستدلال المحلي

    ستضبط هذه في إعدادات وولف فيش بعد التثبيت. التسلسل يجرّب مزودي السحابة أولاً ثم يعود إلى Ollama إن كان يعمل. يمكنك تخطي Ollama كليًا واستخدام مزودي السحابة فقط، أو تخطي السحابة والعمل محليًا بالكامل.

  </Accordion>

  <Accordion title="Ollama (اختياري — يشغّل النماذج المحلية)">
    Ollama يتيح لك تشغيل نماذج مفتوحة المصدر بالكامل على جهازك دون أي اعتماد على الإنترنت. يتكامل وولف فيش معه مباشرة وسيساعدك في اختيار وتنزيل نموذج. لا حاجة لأوامر في الطرفية.

    Ollama **ليس مطلوبًا** — يمكنك استخدام وولف فيش مع مزودي السحابة فقط (DeepSeek أو Claude أو OpenAI). لكن إن أردت العمل دون اتصال أو تفضّل إبقاء كل الاستدلال محليًا، ثبّت Ollama.

    <Note>
      المهام الوكيلية الموثوقة (استخدام أدوات متعدد الخطوات) تتطلب نماذج كبيرة — 70B+ معامل (Llama 3 70B، Qwen 2.5 72B، DeepSeek-V2). تشغيلها يحتاج 48GB+ ذاكرة أو بطاقة GPU متقدمة. النماذج الأصغر (7B–14B) تعمل للمحادثات البسيطة لكنها تكافح مع سير عمل استدعاء الأدوات المعقد. راجع [دليل تكامل Ollama](/ar/configuration/ollama) للتفاصيل.
    </Note>

  </Accordion>

  <Accordion title="أذونات macOS">
    لقدرة computer-use، سيطلب macOS منك منح أذ��نات **الوصول** و**تسجيل الشاشة**. هذه مطلوبة فقط إذا أردت أن يتفاعل وولف فيش مع شاشتك.
  </Accordion>
</AccordionGroup>

## التنزيل

<Tabs>
  <Tab title="macOS">
    نزّل أحدث ملف `.dmg` من [صفحة التنزيل](https://www.wolffi.sh/).

    1. افتح ملف `.dmg`
    2. اسحب وولف فيش إلى مجلد التطبيقات
    3. شغّل وولف فيش من مجلد التطبيقات

    <Note>
      عند التشغيل الأول، قد يحذّر macOS من مطور غير معرَّف. انقر بزر الفأرة الأيمن على التطبيق واختر "فتح" لتجاوز ذلك.
    </Note>

  </Tab>

  <Tab title="Windows">
    نزّل أحدث مثبّت `.exe` من [صفحة التنزيل](https://www.wolffi.sh/).

    1. شغّل مثبّت NSIS
    2. اتبع معالج الإعداد
    3. شغّل وولف فيش من قائمة ابدأ

    جميع الأذونات تُمنح تلقائياً على Windows.

  </Tab>

  <Tab title="Linux">
    نزّل أحدث `.AppImage` من [صفحة التنزيل](https://www.wolffi.sh/).

    ```bash
    chmod +x Wolffish-*.AppImage
    ./Wolffish-*.AppImage
    ```

    <Warning>
      X11 مطلوب. Wayland غير مدعوم حالياً لقدرة computer-use.
    </Warning>

  </Tab>
</Tabs>

## ما المضمّن

لا تحتاج لتثبيت أي شيء آخر. Electron يضمّن Node.js وChromium داخلياً. يأتي التطبيق مع:

- بيئة التشغيل (Node.js + Chromium)
- أكثر من 20 قدرة مدمجة
- SQLite (فهرس البحث FTS5)
- جميع محولات القنوات (سطح المكتب، Telegram، WhatsApp)

## المتطلبات حسب نظام التشغيل

|                       | macOS                           | Windows             | Linux                            |
| --------------------- | ------------------------------- | ------------------- | -------------------------------- |
| **طريقة التثبيت**     | `.dmg` ← سحب للتطبيقات          | مثبّت NSIS (`.exe`) | AppImage (تفعيل الصلاحية، تشغيل) |
| **هل Node.js مطلوب؟** | لا                              | لا                  | لا                               |
| **تبعيات التشغيل**    | لا شيء                          | لا شيء              | لا شيء                           |
| **للذكاء المحلي**     | Ollama (اختياري)                 | Ollama (اختياري)     | Ollama (اختياري)                  |
| **للذكاء السحابي**    | مفتاح API (Anthropic أو OpenAI) | مفتاح API           | مفتاح API                        |
| **الأذونات**          | الوصول + تسجيل الشاشة           | تلقائية             | X11 مطلوب                        |

أدوات النظام مثل `ffmpeg` و`git` **ليست متطلبات مسبقة**. عندما تحتاج قدرة لأداة، يكتشف وولف فيش أنها مفقودة ويعرض تثبيتها لك (عبر مدير حزم نظامك)، بموافقتك. الوكيل يدير التبعيات أثناء التشغيل.

## البناء من المصدر

<Warning>
  البناء من المصدر مخصص للمساهمين فقط. المستخدمون العاديون يجب أن يستخدموا
  المثبّتات الجاهزة أعلاه.
</Warning>

```bash
git clone https://github.com/thewolffish/wolffish-app.git
cd wolffish-app
npm install
npm run dev
```

يتطلب البناء من المصدر Node.js 20+ وnpm. الوحدات الأصلية (better-sqlite3) تُجمّع أثناء `npm install` عبر `electron-rebuild`.

## التحقق من التثبيت

عند التشغيل الأول، ينشئ وولف فيش المجلد `~/.wolffish/workspace/` مع ملفات التكوين الافتراضية. ستوجّهك عملية الإعداد خلال اختيار نموذج والبدء.

للتحقق من أن كل شيء يعمل:

1. اتبع توجيهات الإعداد لاختيار وتنزيل نموذج
2. أرسل رسالة تجريبية مثل "مرحباً، ماذا يمكنك أن تفعل؟"
3. ينبغي أن ترى رداً متدفقاً

إذا لم يُكتشف Ollama، سيعرض عليك وولف فيش إرشادك خلال تثبيته — أو يمكنك التخطي واستخدام مزودي السحابة بدلاً منه.

## إلغاء التثبيت

جميع بيانات المستخدم موجودة في `~/.wolffish/`. لإلغاء التثبيت بالكامل:

1. احذف التطبيق
2. أزل مجلد البيانات: `rm -rf ~/.wolffish/`
