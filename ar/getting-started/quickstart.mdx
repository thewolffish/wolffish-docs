# البدء السريع

يأخذك هذا الدليل من الصفر إلى وكيل يعمل في 5 دقائق.

## 1. أرسل رسالة

اكتب أي شيء في صندوق الدردشة. يجمع Wolffish السياق من ملفات مساحة عملك، ويستدعي نموذج اللغة، ويدفق الرد. جرّب:

```
ما الملفات الموجودة في مجلد المنزل لدي؟
```

إذا كانت قدرات الشل مفعّلة، فسيستدعي الوكيل أداة `shell_exec`، وسترى مربع حوار موافقة للأوامر التي يحتمل أن تكون خطيرة.

## 2. افحص ما حدث

بعد المحادثة، تحقق من هذه الملفات:

| الملف | ما يعرضه |
|---|---|
| `brain/prefrontal/.debug/context-*.md` | المُطالبة الدقيقة التي أُرسلت إلى نموذج اللغة |
| `brain/hippocampus/episodes/YYYY-MM-DD.md` | ملخص المحادثة المحفوظ في الذاكرة |
| `brain/corpus/YYYY-MM-DD.log.md` | كل حدث أُطلق خلال خطّ المعالجة |
| `brain/motor/tasks/TASK-*.md` | سجل خطوة بخطوة لأي عمليات تنفيذ للأدوات |

## 3. أضف قدرة

أنشئ مجلدًا جديدًا في `brain/cerebellum/` يحتوي ملف `SKILL.md`:

```
brain/cerebellum/my-skill/
└── SKILL.md
```

تُسجّل واجهة YAML العلوية للـ SKILL.md القدرةَ في النظام. يحتوي متن Markdown على تعليمات يقرأها نموذج اللغة في وقت التشغيل. راجع [إنشاء القدرات](/ar/extending/creating-capabilities) للدليل الكامل.

## 4. حرّر شخصية وكيلك

استخدم عارض الملفات المدمج في Wolffish لفتح `brain/identity/soul.md` وتعديله مباشرة في التطبيق. يمكنك أيضاً استخدام أي محرر نصوص خارجي. تسري التغييرات على الرسالة التالية — لا حاجة لإعادة التشغيل.

## 5. اضبط المزودين

في الإعدادات، أضف مفاتيح API لمزودي السحابة. ترتيب التدرج هو:

```
Claude → OpenAI → Ollama (محلي)
```

إذا فشل المزود الأساسي، يعود Wolffish تلقائيًا إلى التالي. إذا كنت غير متصل، فسينتقل مباشرة إلى Ollama.
