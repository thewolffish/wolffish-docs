# تحويل الوسائط

يستطيع وولف فيش تحويل أي ملف فيديو إلى صوت MP3 بأمر واحد. أرفق الفيديو، وأخبره بما تريد، وإمكانية ffmpeg المدمجة تتكفل بالباقي — بدون أوامر يدوية، بدون أدوات خارجية.

## الإعداد

### المتطلبات

- **وولف فيش مثبت ويعمل**
- **مفتاح DeepSeek V4 Pro API** — مُعد في الإعدادات > النماذج. سير العمل هذا بسيط بما يكفي لأي نموذج، لكن DeepSeek V4 Pro هو الموصى به — يتعامل مع ترجمة معاملات ffmpeg بموثوقية وبأقل تكلفة.
- **FFmpeg** — الوكيل يتحقق تلقائياً عبر `ffmpeg_check` ويعرض تثبيته عبر `ffmpeg_install` إذا لم يكن موجوداً. إذا أردت تثبيته مسبقاً: `brew install ffmpeg` (ماك) أو `sudo apt install ffmpeg` (لينكس) أو `choco install ffmpeg` (ويندوز).

### لا حاجة لأذونات

يستخدم سير العمل هذا تنفيذ الأوامر وإمكانية ffmpeg فقط. لا حاجة للتحكم بالحاسوب أو المتصفح أو تسجيل الشاشة أو أذونات macOS.

## الأمر

أرفق ملف الفيديو بالمحادثة، ثم أرسل:

```
حوّل هذا الفيديو إلى صوت MP3. استخدم جودة عالية (192kbps).
احفظه في مجلد العمل بنفس اسم الملف لكن بامتداد .mp3.
```

هذا كل شيء. الوكيل يتكفل بالباقي.

### تنويعات

عدّل الأمر للحصول على ما تحتاجه بالضبط:

| ما تريده | الأمر |
|----------|-------|
| **حجم ملف أصغر** | `حوّل هذا الفيديو إلى MP3 بمعدل 128kbps` |
| **أعلى جودة** | `حوّل هذا الفيديو إلى MP3 بمعدل 320kbps` |
| **مقطع فقط** | `استخرج الصوت من 1:30 إلى 3:45 كـ MP3` |
| **اسم ملف مخصص** | `حوّل إلى MP3 واحفظه باسم podcast-episode-12.mp3` |
| **صيغة مختلفة** | `حوّل هذا الفيديو إلى WAV` أو `حوّل إلى FLAC` |
| **تحويل دفعة** | `حوّل جميع ملفات .mp4 في مجلد العمل إلى MP3` |

## كيف يعمل

1. يستقبل الوكيل ملف الفيديو المرفق.
2. يستدعي `ffmpeg_check` للتحقق من تثبيت ffmpeg.
3. إذا لم يكن مثبتاً، يستدعي `ffmpeg_install` لتثبيته عبر مدير الحزم.
4. يستدعي `ffmpeg_run` بالمعاملات المناسبة — شيء مثل:
   ```
   -i input-video.mp4 -vn -acodec libmp3lame -ab 192k output-audio.mp3
   ```
5. يظهر ملف MP3 المحوّل في مجلد العمل.

<Note>
العلامة `-vn` تزيل مسار الفيديو. الوكيل يحدد الترميز ومعدل البت المناسبين من أمرك — لا تحتاج لمعرفة صيغة أوامر ffmpeg.
</Note>

## القيود

- **حجم الملف** — يعالج ffmpeg الملفات على القرص، لذا الحد العملي هو مساحة التخزين المتاحة. حد حماية الذاكرة 500MB ينطبق على مخازن الصوت في الذاكرة، ليس على التحويل المبني على الملفات.
- **دعم الترميز** — يدعم ffmpeg تقريباً كل صيغ الصوت والفيديو. إذا كان ترميز معين مفقوداً، سيحاول الوكيل تثبيت المكتبة المطلوبة.
- **وقت المعالجة** — يتناسب مع طول الملف وجودة المخرجات. فيديو 10 دقائق يُحوّل في ثوانٍ. محاضرة ساعتين تستغرق دقيقة أو دقيقتين. لا يتم تطبيق مهلة — الأمر يعمل حتى ينتهي.

## الأتمتة مع النبض

إذا كنت تستقبل ملفات فيديو بانتظام تحتاج لاستخراج الصوت منها — مثل تسجيلات الاجتماعات أو المحاضرات — يمكنك أتمتة ذلك. افتح **الإعدادات > النبض** وأضف:

```markdown
## Convert New Videos | Daily (09:00)

Check the workspace/recordings folder for any .mp4, .mov,
.webm, or .mkv files that don't already have a matching .mp3.
Convert each one to MP3 at 192kbps and save the .mp3 next to
the original video file. After converting, list what was
processed.
```

<Tip>
غيّر مسار المجلد والجدول ليناسب سير عملك. `Cron (*/30 * * * *)` يعمل كل 30 دقيقة للتحويل شبه الفوري.
</Tip>
