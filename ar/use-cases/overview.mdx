# سيناريوهات واقعية

حالات الاستخدام هي سيناريوهات واقعية توضح ما يمكن لـ وولف فيش فعله من البداية إلى النهاية. كل حالة وصفة كاملة تشمل القدرات المطلوبة، وأمر الـ heartbeat، والخطوات التفصيلية، وما يمكن توقعه.

اتبعها لتستلهم وتتعلم ما يستطيع وولف فيش فعله. كل حالة استخدام تستخدم فقط القدرات المضمَّنة — بدون خدمات طرف ثالث، بدون واجهات API خارجية، بدون كود إضافي. مجرد تعليمات markdown وبحث على الويب وتحكم بالحاسوب.

قبل التعمق في حالة استخدام، اقرأ [التهيئة للنجاح](/ar/use-cases/setting-up-for-success) — تغطي أذونات macOS وأدوات النظام والنموذج الذهني لجعل التشغيلات المستقلة سريعة ورخيصة وموثوقة. وقبل التوسّع في الأتمتة، اطّلع على [القيود والتكاليف](/ar/use-cases/limitations) لإبقاء الإنفاق تحت السيطرة.

| حالة الاستخدام | الوظيفة |
|----------|--------------|
| [استئصال Adobe](/ar/use-cases/nuke-adobe) | يستأصل Adobe Creative Cloud وكل أثر له من جهازك بدقة جراحية وشمول — يعمل على macOS وWindows وLinux، يجرد أولاً، ويترك ملفاتك والمكونات المشتركة دون مساس، وينتهي بتقرير مصنَّف |
| [التفاعل على Reddit](/ar/use-cases/reddit-engage) | يبحث ويصيغ وينشر 5 منشورات Reddit مُحسَّنة للانتشار عبر 5 مجتمعات subreddit من خلال جلسة متصفحك المسجَّل دخولها — مستقل بالكامل، بدون Reddit API |
| [حصد الكارما](/ar/use-cases/karma-farm) | ينشر 5 ردود عالية القيمة عبر مواضيع subreddit شائعة لكسب الكارما من خلال جلسة متصفحك المسجَّل دخولها — مستقل بالكامل، بدون Reddit API |
| [تحويل الوسائط](/ar/use-cases/media-conversion) | يحوّل ملفات الفيديو إلى صوت MP3 باستخدام إمكانية ffmpeg المدمجة — أرفق فيديو واحصل على ملف صوتي |
| [منشور LinkedIn اليومي](/ar/use-cases/ai-news-linkedin-post) | يبحث عن أخبار الذكاء الاصطناعي اليوم ويكتب منشور تجميع ويفتح Chrome ويكتبه وينشره على LinkedIn — مستقل بالكامل |
| [تدقيق أمني](/ar/use-cases/mac-security-audit) | يدقّق المنافذ المفتوحة، والجدار الناري، والخدمات، وإعداد SSH، وتشفير القرص، والبرامج القديمة ذات الثغرات المعروفة (CVEs) — يعمل على macOS وLinux وWindows |
