# Real-World Workflows

Use cases are real-world workflows that show what Wolffish can do end to end. Each one is a complete recipe — the capabilities it needs, the heartbeat prompt, the step-by-step flow, and what to expect.

Follow them to get inspired and learn what Wolffish is capable of. Every use case uses only built-in capabilities — no third-party services, no external APIs, no extra code. Just markdown instructions, web search, and computer use.

Before diving into a use case, read [Setting Up for Success](/use-cases/setting-up-for-success) — it covers macOS permissions, system tools, and the mental model for making autonomous runs fast, cheap, and reliable.

| Use Case | What It Does |
|----------|--------------|
| [Setting Up for Success](/use-cases/setting-up-for-success) | Permissions, prerequisites, and tips that apply to every use case |
| [Limitations & Costs](/use-cases/limitations) | How automation costs accumulate, model selection, expected costs, and recommendations for keeping spend under control |
| [Media Conversion](/use-cases/media-conversion) | Converts video files to MP3 audio using the built-in ffmpeg capability — attach a video, get an audio file |
| [Gmail Is Dead](/use-cases/gmail-is-dead) | Connects your Gmail via OAuth, reads and summarizes unread emails, drafts replies, and manages your inbox end to end |
| [PDFs Everywhere](/use-cases/pdfs-everywhere) | Generates professional, research-backed PDFs on demand — news digests, competitor reports, travel guides, and more |
| [CV Tailor](/use-cases/cv-tailor) | Takes your existing CV and a job description, generates a new CV tailored to that specific role |
| [Security Audit](/use-cases/mac-security-audit) | Audits open ports, firewall, services, SSH, disk encryption, and outdated software with known CVEs — works on macOS, Linux, and Windows |
| [AI News Daily LinkedIn Post](/use-cases/ai-news-linkedin-post) | Searches for today's AI news, writes a roundup post, opens Chrome, types and publishes it to LinkedIn — fully autonomous |
