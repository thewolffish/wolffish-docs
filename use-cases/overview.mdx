# Real-World Workflows

Use cases are real-world workflows that show what Wolffish can do end to end. Each one is a complete recipe — the capabilities it needs, the heartbeat prompt, the step-by-step flow, and what to expect.

Follow them to get inspired and learn what Wolffish is capable of. Every use case uses only built-in capabilities — no third-party services, no external APIs, no extra code. Just markdown instructions, web search, and computer use.

Before diving into a use case, read [Setting Up for Success](/use-cases/setting-up-for-success) — it covers macOS permissions, system tools, and the mental model for making autonomous runs fast, cheap, and reliable. And before you automate at scale, skim [Limitations & Costs](/use-cases/limitations) to keep spend under control.

| Use Case | What It Does |
|----------|--------------|
| [Product Hunter](/use-cases/product-hunter) | Browses Product Hunt through your logged-in browser extension, tracks every top launch from the past week, researches the company, founders, and funding behind each one, reads the community consensus and top comments, and delivers a ranked, categorized PDF with stats, links, and an executive summary |
| [News Butler](/use-cases/news-butler) | Exhaustively scans the public web for everything that happened this week across the topics you care about, cross-checks every claim against other outlets, profiles each source, and delivers one clean, color-coded PDF with an executive summary, easy numbers, and charts |
| [Nuke Adobe](/use-cases/nuke-adobe) | Surgically and exhaustively removes Adobe Creative Cloud and every trace of it from your machine — works on macOS, Windows, and Linux, inventories first, leaves your files and shared components untouched, ends with a classified report |
| [Engage Reddit](/use-cases/reddit-engage) | Researches, drafts, and publishes 5 viral-optimized Reddit posts across 5 subreddits through your logged-in browser session — fully autonomous, no Reddit API |
| [Karma Farm](/use-cases/karma-farm) | Posts 5 high-value replies across popular subreddit threads to earn karma through your logged-in browser session — fully autonomous, no Reddit API |
| [Media Conversion](/use-cases/media-conversion) | Converts video files to MP3 audio using the built-in ffmpeg capability — attach a video, get an audio file |
| [Gmail Is Dead](/use-cases/gmail-is-dead) | Connects your Gmail via OAuth, reads and summarizes unread emails, drafts replies, and manages your inbox end to end |
| [PDFs Everywhere](/use-cases/pdfs-everywhere) | Generates professional, research-backed PDFs on demand — news digests, competitor reports, travel guides, and more |
| [CV Tailor](/use-cases/cv-tailor) | Takes your existing CV and a job description, generates a new CV tailored to that specific role |
| [Security Audit](/use-cases/mac-security-audit) | Audits open ports, firewall, services, SSH, disk encryption, and outdated software with known CVEs — works on macOS, Linux, and Windows |
| [Daily LinkedIn Post](/use-cases/ai-news-linkedin-post) | Searches for today's AI news, writes a roundup post, opens Chrome, types and publishes it to LinkedIn — fully autonomous |
